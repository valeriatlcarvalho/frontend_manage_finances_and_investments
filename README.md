# frontend_manage_finances_and_investments

Gerencie finanças e investimentos pessoais

## Instalar ambiente

Se não tiver o `fnm` instalado ainda, pode instalá-lo pelo comando:

```shell
choco install fnm
```

Versão do `npm`

```shell
$ npm -v
11.0.0
```

Versão do `node`

```shell
$ node -v
v22.13.1
```
